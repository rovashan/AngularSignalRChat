# AngularSignalRChat

A chat application that showcases websocket broadcast communication using ASP.NET SignalR and Angular.

## Prerequisites
-  .NET Core 3.1
-  Angular 7

## How to run

### Angular client
```sh
npm install
ng serve
```

### SignalR server
```sh
dotnet run
```

## Features
- Login page to capture nickname.
- Dynamic avatar generation using https://robohash.org.
